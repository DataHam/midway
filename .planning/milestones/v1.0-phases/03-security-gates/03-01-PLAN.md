  <action>
Create shared rate limiting module with:

1. **checkRateLimit function**:
   - Reads session cookie from request (cookie name: "tamtham_rate_limit")
   - Parses cookie value: { attempts: number, timestamp: ISO string }
   - Returns: { allowed: boolean, remaining: number }
   - If no cookie exists: allowed = true, remaining = 3
   - If attempts >= 3: allowed = false, remaining = 0
   - Cookie expires after 1 hour from first attempt

2. **recordFailedAttempt function**:
   - Increments attempt counter in cookie
   - Sets cookie with 1-hour expiry
   - Returns updated response with Set-Cookie header
   - Cookie structure: "tamtham_rate_limit={attempts:3,timestamp:ISO}; Path=/; Max-Age=3600"

3. **Cookie security**:
   - HttpOnly flag (not accessible to JavaScript)
   - SameSite=Strict (CSRF protection)
   - Secure flag (HTTPS only in production)

Reference locked decisions: 3 attempts per session, cookie-based tracking (not IP-based), modal error display for RATE_LIMITED.
