  <name>Task 1: Create staticwebapp.config.json with security headers</name>
  <files>staticwebapp.config.json</files>
  <action>
    Create staticwebapp.config.json with the following structure:

    1. **globalHeaders** section with 5 security headers:
       - `strict-transport-security`: "max-age=31536000; includeSubDomains; preload"
       - `content-security-policy`: Allow self, Turnstile (challenges.cloudflare.com), self-hosted fonts/images
       - `x-content-type-options`: "nosniff"
       - `x-frame-options`: "DENY"
       - `referrer-policy`: "strict-origin-when-cross-origin"

    2. **routes** section with cache-control headers:
       - `/pages/*`: public, max-age=3600 (HTML pages)
       - `/assets/fonts/*`: public, max-age=31536000, immutable (WOFF2)
       - `/assets/images/*`: public, max-age=31536000, immutable (WebP)
       - `/favicon.ico`: public, max-age=31536000, immutable

    3. **navigationFallback** for SPA routing:
       - rewrite: "/pages/index.html"

    CSP details:
    - default-src: 'self'
    - script-src: 'self' 'unsafe-inline' https://challenges.cloudflare.com (Turnstile)
    - style-src: 'self' 'unsafe-inline' (Tailwind + self-hosted)
    - font-src: 'self' (Montserrat WOFF2)
    - img-src: 'self' data: (WebP images + inline data URIs)
    - frame-src: https://challenges.cloudflare.com (Turnstile iframe)
    - connect-src: 'self' (API calls)
    - object-src: 'none' (no plugins)
    - base-uri: 'self'
    - form-action: 'self'

    Reference: 04-SECURITY-SEO-RESEARCH.md, Pattern 1: Security Headers Configuration
  </action>
