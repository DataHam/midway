<task type="auto">
  <name>Task 2: Create robots.txt with sitemap reference</name>
  <files>robots.txt</files>
  <action>
    Create robots.txt with standard crawl directives:

    1. **Comment line**: `# Tam-Tham Website robots.txt`
    2. **User-agent**: `*` (all crawlers)
    3. **Allow directive**: `Allow: /` (allow all paths)
    4. **Sitemap reference**: `Sitemap: https://tamtham.com/sitemap.xml`

    Do NOT include:
    - Disallow rules (all content should be indexed)
    - Crawl-delay (not needed for this site)
    - Specific user-agents (use wildcard *)

    Reference: 04-SECURITY-SEO-RESEARCH.md, Pattern 4: robots.txt Configuration
  </action>
  <verify>
